# Customisable Budget App

A personal finance app in GBP (£) where users sign in, track money in and out in real time, set savings goals, store recurring bills and income, and log shopping on the go. Installable on phone as a PWA.

## Core features

**Authentication**
- Email/password + Google sign-in
- Each user's data is private (secured at the database level)
- Sign up, sign in, sign out, password reset

**Dashboard (balanced overview)**
- Current month balance: income vs outgoings, money left
- Donut chart: spending by category
- Bar chart: last 6 months trend
- Savings goals progress bars
- Upcoming recurring bills (next 7 days)
- Recent transactions feed
- Everything updates in real time as you add entries — no refresh needed

**Income**
- One-off income entries (date, amount, source, note)
- Recurring income streams (e.g. salary £2,400 monthly on the 28th) — auto-generated each cycle
- Edit, pause, or delete recurring streams

**Outgoings**
- One-off outgoings with category (Rent, Utilities, Food, Transport, Entertainment, Health, Other — customisable)
- Recurring bills (e.g. Netflix £10.99 monthly on the 5th) — auto-generated each cycle
- Edit, pause, or delete

**Shopping (two modes)**
- *Quick mode*: one tap — amount, category (defaults to Food), optional note
- *Itemised mode*: add items line by line (Bread £2, Milk £1.50…) with running total, save as one shop
- Toggle between modes on the entry screen

**Savings goals**
- Create goals (name, target amount, target date, optional icon/colour)
- Add contributions manually, or link a recurring transfer
- Progress bar + projected completion date based on contribution pace

**Categories**
- Default set provided, fully customisable: rename, add, delete, recolour
- Per-category monthly budget limits with warnings when exceeded

**Settings**
- Currency locked to GBP (£) by default — easy to swap later if you want
- Profile, notification preferences, data export (CSV)

## Mobile / PWA

- Fully responsive, mobile-first design
- Installable via "Add to Home Screen" on iOS and Android — works like a native app icon
- Custom app icon and splash screen
- Note: PWA install behaviour only works fully in the published version, not in the in-editor preview

## UI direction

- Clean, modern, friendly — soft cards, rounded corners, generous spacing
- Light + dark mode
- Bottom tab bar on mobile (Dashboard / Add / Goals / Settings), sidebar on desktop
- Big floating "+" button to log income, outgoing, or shopping in two taps
- Smooth charts and progress animations

## Build approach

I'll build it in stages so you can review as we go:

1. **Foundation** — Auth (email + Google), database schema, navigation shell, dashboard skeleton
2. **Core tracking** — Income & outgoings (one-off), categories, real-time dashboard with charts
3. **Recurring + goals** — Recurring bills/income engine, savings goals with progress
4. **Shopping** — Quick + itemised shopping flows
5. **Polish + PWA** — Mobile UX refinements, dark mode, PWA manifest & install prompt, CSV export

## Technical notes

- Lovable Cloud for auth, database, real-time subscriptions
- Row-level security so each user only sees their own data
- Recurring entries generated by a scheduled job that runs daily
- Real-time updates via Supabase Realtime channels on transactions table
- PWA via web app manifest with `display: standalone` (no service worker, to keep preview reliable)
